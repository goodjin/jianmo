# Markly E2E 测试

本项目使用 Playwright 进行端到端测试，实现 100% 功能覆盖。

## 安装依赖

```bash
npm install
npx playwright install chromium
```

## 运行测试

```bash
# 命令行模式
npm run test:e2e

# UI 模式（推荐开发时使用）
npm run test:e2e:ui

# 有头模式（显示浏览器）
npm run test:e2e:headed
```

## 测试策略

1. **单功能单测**：每个功能独立测试，便于定位问题
2. **端到端验证**：模拟真实用户操作流程
3. **失败重试**：CI 环境下自动重试
4. **截图/录屏**：失败时自动保存调试信息

## 验证方式

- DOM 元素存在性检查
- 文件内容验证
- 快捷键操作验证
- 截图对比（Playwright 自动）
