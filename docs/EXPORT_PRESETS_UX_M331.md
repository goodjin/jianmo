# 导出：预设/配置 UX（可选）（M331）

## 结论（本阶段）

**延期实现，仅固化口径与最小设计**。原因：当前导出链路已经具备预检、失败诊断、自包含安全策略与主题扩展点；预设 UI 属于“锦上添花”，不应阻塞封板门禁。

## 最小设计（未来实现参考）

- **预设实体**：`name` + `format`(pdf/html) + `themeId` + `options`
- **作用域**：
  - 默认：workspace（可跟随项目）
  - 可选：user（跨项目）
- **入口**：
  - 导出面板：Preset 下拉 + “管理预设”
  - 命令：`Markly: Export (Preset...)`

## 兼容与自救

- 预设读取失败不阻塞导出：回落到“当前默认配置”
- 导出失败仍走统一诊断 UX（见 `docs/EXPORT_FAILURE_DIAGNOSTICS_UX_M324.md`）
