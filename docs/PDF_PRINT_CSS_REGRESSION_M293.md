# PDF 打印 CSS 回归集（M293）

> 目的：把 PDF 导出中最常见的排版回归（CJK、代码块换行、表格、字体回退）形成固定检查清单与单测锚点。

## 回归项清单

- **CJK 字体回退**：中文标题/正文不应显示为方框；中英文混排行高稳定。
- **代码块换行**：长行不应把版心撑爆；允许换行但不应丢字符。
- **表格**：表头样式稳定；宽表不应溢出页面（必要时可横向滚动或换行策略固定）。
- **页眉页脚**：开启时不应覆盖正文；页码/标题模板渲染稳定。

## 工程锚点

- 实现：`src/core/export/pdfExport.ts`（模板 CSS + Puppeteer `page.pdf`）
- 门禁：`src/core/export/__tests__/pdfExport.test.ts`

## 手工验证（发布前）

建议用包含以下内容的文档导出一次 PDF：

- 中文段落（含标点、全角空格）
- 超长代码行（含 URL/路径）
- 20+ 行表格
- 数学公式与 Mermaid（如启用）
