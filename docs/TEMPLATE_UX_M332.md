# 模板：用户模板体验增强（可选）（M332）

## 结论（本阶段）

**不改现有实现，只补“用户体验口径 + 增强路线”**。原因：模板体系已能满足基本可用（内置模板 + 用户目录），本阶段优先保证门禁与发布稳定性。

## 当前能力

- 内置模板：随扩展发布（见 `templates/`）
- 用户模板目录：由扩展提供入口/规则（单测已覆盖）

## 未来增强建议（不破坏兼容）

- **模板预览**：在选择模板时显示预览摘要（标题、适用场景、截图）
- **模板变量**：支持 `{title}` / `{date}` 等轻量变量替换（仅作用在新建文档，不做全局宏）
- **模板分享**：导出单个模板为 zip（不自动下载第三方）

## 失败策略

- 读取用户模板失败：回落到内置模板；并在诊断里给出路径与错误原因
