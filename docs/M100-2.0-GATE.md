- 扩展 ↔ webview **消息结构**不兼容旧版，且无兼容层。

若本周期仅有**新增能力、默认行为不变、协议向后兼容**，则一般 **MINOR / PATCH** 即可（与 `CLAUDE.md` 版本规则一致）。
