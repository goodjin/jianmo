# 用户沟通：迁移说明/FAQ（IR 已移除）（M348）

## 一句话总结

Markly 不再提供 IR（中间态/装饰模式）。你仍可以在 **Rich** 与 **Source** 之间切换，编辑与导出不受影响。

## 常见问题

### 1) 我原来用 IR，现在怎么办？

- **推荐**：用 **Rich**（所见即所得）
- 若你需要纯文本手感或极端兼容：用 **Source**

### 2) 升级后打开是白屏/红屏怎么办？

1. 先执行 `Developer: Reload Window`
2. 打开 **Recovery & Troubleshooting**，点击“复制诊断信息”
3. 临时自救：切到 **Source**，先把内容编辑/保存下来

### 3) 我的快捷键失效了？

看 `docs/SHORTCUT_CONFLICTS_FAQ_M329.md`，可在 VS Code “键盘快捷方式”里重新绑定。
