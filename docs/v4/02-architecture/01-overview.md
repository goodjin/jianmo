## 1. 系统定位

### 1.1 系统概述

Markly v4 是基于 CodeMirror 6 构建的 VS Code Markdown 编辑器扩展，采用装饰模拟 WYSIWYG 架构，提供即时渲染、源码编辑和分屏预览三种模式。

**核心特征**:
- **大模型友好**: 架构清晰，状态可预测，便于 AI 辅助开发
- **高性能**: 利用 CodeMirror 6 虚拟滚动，支持 10MB+ 大文档
- **可测试**: 纯函数状态管理，易于单元测试
- **可扩展**: 模块化 Extension 架构，功能可独立开发和复用

## 5. 关键设计决策

### 5.1 状态管理策略

**决策**: 采用 CodeMirror 6 原生状态管理 + Vue 3 响应式系统组合

```typescript
// 状态分层设计
interface AppState {
  // Layer 1: CodeMirror 原生状态 (不可变)
  editor: {
    state: EditorState;           // CodeMirror EditorState
    view: EditorView;             // CodeMirror EditorView
  };

  // Layer 2: 应用层状态 (Vue 3 reactive)
  app: {
    mode: 'ir' | 'source' | 'split';  // 当前模式
    theme: 'light' | 'dark' | 'auto'; // 当前主题
    config: DocumentConfig;           // 文档配置
  };

  // Layer 3: 派生状态 (computed)
  derived: {
    headings: HeadingNode[];      // 大纲树
    wordCount: number;            // 字数统计
    charCount: number;            // 字符统计
  };
}
```

**理由**:
1. CodeMirror 6 状态是不可变的，符合函数式编程理念
2. Vue 3 响应式系统与 CodeMirror 的 updateListener 配合良好
3. 分层清晰，便于测试和调试

### 5.2 装饰器渲染策略

**决策**: 使用 CodeMirror 6 ViewPlugin + Decoration 实现 WYSIWYG

```typescript
// 装饰器架构
const decoratorArchitecture = {
  // 基础装饰器
  base: [
    'headingDecorator',      // 标题样式
    'emphasisDecorator',     // 粗体/斜体
    'linkDecorator',         // 链接
    'codeDecorator',         // 行内代码
  ],

  // 交互装饰器
  interactive: [
    'taskListDecorator',     // 任务列表复选框
    'mathDecorator',         // 数学公式
  ],

  // 渲染策略
  strategy: {
    // 即时渲染模式：显示装饰
    ir: 'all',
    // 源码模式：仅语法高亮
    source: 'none',
    // 分屏模式：编辑区语法高亮，预览区完整渲染
    split: 'minimal',
  }
};
```

**理由**:
1. 装饰系统是 CodeMirror 6 推荐的可视化方案
2. 比 ProseMirror 的节点替换更简单可靠
3. 性能更好，支持虚拟滚动

### 5.3 与 VS Code 通信策略

**决策**: 使用 VS Code Webview API 的 postMessage 进行双向通信

```typescript
// 通信协议
interface VSCodeMessage {
  type: 'init' | 'update' | 'save' | 'config' | 'image';
  payload: unknown;
}

// 通信流程
const communicationFlow = {
  // Webview → Extension
  webviewToExtension: [
    'CONTENT_CHANGE',    // 内容变更
    'SAVE_REQUEST',      // 保存请求
    'IMAGE_UPLOAD',      // 图片上传
    'CONFIG_UPDATE',     // 配置更新
  ],

  // Extension → Webview
  extensionToWebview: [
    'INIT_CONTENT',      // 初始化内容
    'CONFIG_CHANGE',     // 配置变更
    'IMAGE_SAVED',       // 图片保存完成
    'THEME_CHANGE',      // 主题变更
  ],
};
```
