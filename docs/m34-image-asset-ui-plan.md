# M34 图片资产 UI 收口计划

目标：把 M32/M33 已完成的图片检测、定位、修复能力从命令面板推进到编辑器可见 UI，让用户一眼知道缺失图片数量，并能直接复制清单、打开目录、修复引用。

## M34-0：计划与验收标准

- [x] 新建本计划文档
- [x] 明确首批范围：不引入复杂侧栏，优先使用提示条和轻量弹窗
- [x] 验收：用户不需要打开命令面板，也能发现和处理缺失图片

## M34-1：缺失图片数量提示

- [x] 在编辑器顶部增加图片资产提示条
- [x] 显示缺失本地图片引用数量
- [x] 复用已有本地图片检测结果，避免新增频繁文件扫描

## M34-2：图片资产操作入口

- [x] 在提示条加入复制缺失清单入口
- [x] 在提示条加入打开 assets 目录入口
- [x] 在提示条加入修复第一项入口

## M34-3：可操作提示闭环

- [x] 缺失图片时提供直接可点的处理按钮
- [x] 按钮复用 M33 的复制、目录、修复逻辑
- [x] 无缺失时继续沿用 toast 告知用户

## M34-4：轻量图片资产面板

- [x] 新增轻量弹窗展示缺失图片列表
